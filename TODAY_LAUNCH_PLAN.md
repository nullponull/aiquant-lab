## 連載開始週のリプライ対応

### 想定される質問 + 返答テンプレ

**Q: GitHub のスター付けるべき？**
```
ありがとうございます！スターは連載継続の励みになります。
コードへのフィードバックも歓迎です。
```

**Q: 投資は実際にしているの？**
```
実取引は行っていません。すべて公開市場データに基づく
仮想シミュレーションです。連載の透明性を最優先しているため。
```

**Q: API キーが必要ですか？**
```
はい、第2回以降の LLM 実験には Anthropic API キーが必要です。
$5 程度の入金で十分です。
第1回のバックテストは API キーなしで実行できます。
```

**Q: なぜ note と GitHub 両方？**
```
note は読みやすい記事用、GitHub はコードと再現性のため。
すべての検証は誰でも追試できる透明性を維持したいので。
```
