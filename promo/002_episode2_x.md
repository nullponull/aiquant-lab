### コスト計算機ツイート

```
LLM議論型エージェントのコスト概算:

API call/event = N × R
(N=エージェント数, R=ラウンド数)

30イベントの実験コスト:
N=3, R=1: $0.12
N=5, R=1: $0.20
N=3, R=2: $0.25
N=10, R=1: $0.40

実装する前に計算した方がいい指標。
```

### Evaluator型推奨ツイート

```
今回の実験で再確認:

LLMは「仮説生成」に強く
「評価」には弱い。

Generator → 仮説生成（LLM）
Evaluator → 数値評価（決定論的）

この分離設計が、議論型より:
✓ コスト1/N
✓ 再現性高い
✓ バックテスト可能

```
