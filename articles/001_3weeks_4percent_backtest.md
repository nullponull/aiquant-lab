# 「年利換算101%のAI自動売買」戦略を10年回したら、SPYとほぼ同じだった件

― あるバズったツイートの戦略を真面目にバックテストしてみた

---

## はじめに

少し前、Xで「Claude Codeで株式投資を自動化したら3週間で+4.19%勝った。月利換算で約6%、年利換算では約101%になる」というポストが4万閲覧を集めて話題になった。

私自身、AIエージェントで投資自動化システムを設計している立場として、このポストにはいくつか気になる点があった。否定したいわけでも本人を批判したいわけでもない。ただ純粋に、**この戦略を10年動かしたら何が起きるのか**が知りたかった。

そこで実際にコードを書いて、過去10年（2014-2024）のデータでバックテストしてみた。結論からいうと、**戦略はちゃんと機能している。ただし、ほぼS&P500を買っているのと同じ結果になった**。

全コードはGitHubで公開している。誰でも追試できる。

---

## 検証対象の戦略

元ポストで言語化されていた戦略は、要約するとこうだ：

| 配分 | 内容 | ルール |
|------|------|--------|
| 70% | 配当貴族 | 長期保有、損切りなし |
| 30% | 中期成長株 | 8%損切り / 20%利確 |

「絶対に負けない企業に投資しつつ、期待値が合っているなら短期も回す」という考え方。**論理は明快で、初心者がここまで言語化できるのは正直すごい**と思う。

ただ気になった点が3つある。

### 気になった点1: 3週間という検証期間

3週間は約15営業日。S&P500の日次リターン標準偏差は約1.2%なので、3週間の標準偏差は√15 × 1.2% ≈ **4.6%**。

つまり、**何もしなくても3週間で±4%程度の振れは普通に起きる**。本人の+4.19%が「戦略の実力」なのか「ノイズの範囲」なのかは、3週間では区別がつかない。

これは後ほど数値で示す。

### 気になった点2: 期待値計算

ポストでは「(0.33 × 20%) + (0.67 × -8%) = 1.2%」「3回に1回勝てばトントン」という計算が示されていた。

実際に計算すると：
- EV = 0.33 × 20 + 0.67 × -8 = 6.6 - 5.36 = **+1.24%**
- これは「トントン」ではなく「微益」
- 真の損益分岐点は勝率 8/(8+20) = **28.6%**

ここは細かい話なので大したことではない。ただ、**手数料・スリッページ・税金（米国株配当には10%源泉徴収）・為替変動を含めると、実際の損益分岐点はもう少し上がる**。

### 気になった点3: 年利換算

「(1.06)^12 - 1 = 約101%」という計算。これは数学としては正しいが、3週間の結果を年換算する統計的根拠はない。

**サイコロ1回で6が出たから「将来の出目を100%6と予測する」と言っているのと同じ**。短期サンプルからの外挿は最も典型的な統計的誤りで、機関投資家の世界では「複利の罠」と呼ばれて警戒されている。

## 実装と検証

ここからが本題。実際にバックテストする。

### セットアップ

- 期間: 2014年1月 〜 2024年12月（約11年）
- 初期資金: 100万円
- 配当貴族プロキシ: NOBL（ProShares S&P 500 Dividend Aristocrats ETF）
- 中期成長株プロキシ: QQQ（NASDAQ-100 ETF）
- 評価指標: 累積リターン、CAGR、シャープレシオ、最大ドローダウン

なお、本人ポストでは個別銘柄選定にも触れていたが、選定ロジックが明示されていなかったため、**ここでは「中期成長株 = QQQ」という前提で実装した**。これにより条件は本人にとってむしろ有利（QQQはこの10年で年率20%超のパフォーマンスを出している）。

### コード抜粋

```python
def simulate_strategy(prices, initial_capital=1_000_000):
    div_alloc = initial_capital * 0.70  # 配当貴族
    growth_alloc = initial_capital * 0.30  # 中期成長

    # 配当貴族: NOBL バイアンドホールド
    nobl_units = div_alloc / prices["NOBL"].iloc[0]

    # 中期成長: -8%/+20%ルール
    growth_position = None
    for date, row in prices.iterrows():
        if growth_position is None:
            growth_position = enter(row["QQQ"], growth_cash)
        else:
            ret = (row["QQQ"] - growth_position["entry"]) / growth_position["entry"]
            if ret <= -0.08 or ret >= 0.20:
                growth_cash = exit_position(growth_position, row["QQQ"])
                growth_position = enter(row["QQQ"], growth_cash)
```

## 結果

### 11年累計

| 戦略 | 最終評価額 | 累積リターン | CAGR | シャープレシオ | 最大DD |
|------|----------|-----------|------|------------|-------|
| **ツイート戦略（70/30）** | ¥3,937,497 | +293.7% | **13.29%** | 0.83 | -32.3% |
| **SPYバイアンドホールド** | ¥3,902,329 | +290.2% | **13.20%** | 0.81 | -33.7% |
| **NOBL単独（配当貴族のみ）** | ¥2,846,989 | +184.7% | 9.99% | 0.67 | -35.4% |

**結論**: 戦略のCAGR 13.29%、SPYバイアンドホールドのCAGR 13.20%。**差は0.09%**。

つまり、**この戦略を10年動かして得られる成果は、SPYを買って完全に放置していた場合とほぼ同じ**だった。手数料を考慮すれば、むしろSPYの方が有利になる可能性が高い。

### 年次リターン比較

```
        戦略       SPY
2015   +3.2%    +1.2%
2016  +10.2%   +12.0%
2017  +24.7%   +21.7%
2018   -2.2%    -4.6%
2019  +31.4%   +31.2%
2020  +22.9%   +18.3%   ← コロナ年でも戦略の方が良い
2021  +26.3%   +28.7%
2022  -18.0%   -18.2%   ← 利上げで両者揃って沈む
2023  +25.1%   +26.2%
2024  +15.5%   +25.3%   ← SPYに大きく劣後
```

戦略はSPYに勝つ年もあれば負ける年もある。**統計的には「ほぼ同じ動き」と言ってよい**。

### 3週間で+4%以上を達成する確率

ここが今回の検証で最も重要な数値だ。

過去11年、SPYを買って何もせず保有し続けた場合の「任意の3週間（15営業日）リターン」の分布を取った。

```
平均: +0.82%
中央値: +1.25%
標準偏差: 3.76%
+4%以上を達成した割合: 13.2%
```

つまり、**SPYを買って何もしなければ、約7-8回に1回は3週間で+4%以上勝てる**。これは戦略の優劣ではなく、市場のノイズが生み出す現象だ。

「3週間で+4%」という結果は、戦略の優位性を示す証拠としては**完全に弱い**。

### 取引統計

戦略の中期成長部分について：
- 総取引数: 24回（10年で）
- 利確: 15回 / 損切り: 9回
- 勝率: 62.5%
- 1取引あたり平均リターン: +9.29%

数字だけ見ると素晴らしい勝率だ。ただしこれは**QQQがこの10年で年率20%超の上昇トレンドだった**ことが大きい。同じルールを2000-2010年（ITバブル崩壊+リーマンショック）で動かすと、勝率は逆転する可能性が高い。

## では何が言えるのか

### 戦略は「悪くない」

これは強調しておきたい。配当貴族70% + 成長株ルール30%という設計は、**論理的に破綻していない**し、長期で見て市場平均と並走する程度の力はある。初心者が3週間でここまで形にできるのは、AI支援の威力を示す例として価値がある。

### ただし「市場平均を超える」とは言えない

10年バックテストではSPYと差がほぼゼロ。これは戦略が悪いのではなく、**ある意味当然の結果**だ。なぜなら：

1. 70%は配当貴族（=高配当大型株）= 大型株インデックスとほぼ連動
2. 30%は損切り/利確を入れているが、トレンド市場では結局リトライしてエントリーし直すので長期保有とほぼ同じ結果
3. 結果として「ほぼS&P500を持っている状態」と同等

これは戦略のロジックを丁寧に追えば、最初から予測できる結論でもある。

### 「年利101%」は完全に過大評価

3週間の結果を年換算するのは、統計的には根拠がない。10年バックテストで判明したCAGRは13%強。**真の年利期待値は10-15%程度**で、これは市場平均と一致する。

「101%」と「13%」では7倍以上の乖離がある。この差は戦略の問題ではなく、**短期サンプルから無理に外挿した数学的な錯覚**だ。

### 「絶対負けない企業」は存在しない

10年バックテストでは2022年に -18% のドローダウン、累計でも -32% の最大ドローダウンを記録した。配当貴族70%でも、利上げ局面ではこのレベルの下落は避けられない。

## 「負けないシステム」の本当の作り方とは

ここまで批判的に見てきたが、**ではどうすれば本当に「負けにくい」AI投資システムが組めるのか**？

これは長くなるので別記事で書く。要点だけ列挙すると：

1. **複数戦略の分散**: 単一戦略では市場レジーム変化に対応できない
2. **数値評価の徹底**: LLM同士の議論は無意味、過去データとの照合がコア
3. **ウォークフォワード検証**: バックテスト一発勝負は過学習を生む
4. **規模感の理解**: 個人で年30%以上を狙うと逆に破綻しやすい
5. **失敗の記録**: 勝った話より負けた話を残す方が実用知が増える

これらを実装した「**100戦略並列稼働シミュレーション**」を、来週から公開で始める。100万円の仮想ポートフォリオで、6ヶ月かけて何が機能して何が機能しないかを記録していく。

| 項目 | 評価 |

派手な数字を煽るのは簡単だが、本当に重要なのは長期での再現性だ。
