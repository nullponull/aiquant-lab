# Phase 1: 法的クリア確認の手順

> Phase 0 で抽出した `viable` 候補について、量産・販売に進む前の法的リスク確認

---

## 確認すべき 4 つの権利

期限切れ特許でも、以下が生きている場合は使えない:

| 権利 | 期間 | 確認方法 |
|------|------|--------|
| **特許権** | 出願から 20 年（医薬品等の例外あり） | J-PlatPat 「経過情報」タブ |
| **意匠権** | 出願から 25 年（旧法では 20 年） | J-PlatPat 「意匠検索」 |
| **商標権** | 10 年ごと更新可（無期限延長可能） | J-PlatPat 「商標検索」 |
| **改良発明** | 元特許とは別の特許 | 同じ発明者・出願人で再検索 |

---

## チェックリスト (各候補 1 件あたり)

```
□ J-PlatPat で経過情報を確認
   □ ステータスが「権利消滅」「期間満了」「年金未納による消滅」のいずれか
   □ 復活手続きが取られていないか
   □ 補正・分割出願の有無

□ 意匠検索
   □ 同じ製品形状で意匠登録されていないか
   □ 関連意匠が現役か

□ 商標検索
   □ 製品名/ブランド名が商標登録されているか
   □ 似た商標で別商品の登録がないか

□ 改良発明検索
   □ 同じ発明者で別の特許が現役でないか
   □ 同じ出願人で別の特許が現役でないか
   □ 引用関係から派生特許がないか

□ 米国・中国対応特許の確認
   □ 同じ発明が他国で保護されていないか (Espacenet で確認)
   □ Amazon US / 中国市場で同様製品の特許訴訟事例

□ 第三者の権利
   □ 「公知技術」として誰でも使えると主張できる証拠がある
   □ 製品形状が業界の慣習と被らない
```

## 弁理士への依頼テンプレート

```
件名: 期限切れ特許の事業化前 権利クリア確認依頼

[弁理士事務所名] 御中

期限切れ特許の事業化を検討しており、以下の観点からの権利クリア
確認を依頼したく存じます。

【対象特許】
特許番号: JP2003-XXXXXXA
タイトル: ○○
公開日: YYYY-MM-DD
出願人: ○○

【確認依頼事項】
1. 当該特許権の現在のステータス (権利消滅・期間満了・年金未納等)
2. 同じ発明者/出願人による関連特許 (改良発明・分割出願) の有無
3. 同じ製品形状・名称の意匠権・商標権の登録有無
4. 米国・中国の対応特許の有無
5. 当該技術領域で第三者の特許訴訟が行われた事例の有無

【背景】
当社は J-PlatPat および Google Patents で予備調査を行い、
権利消滅していると認識しています。
量産・販売に進む前に、専門家のお墨付きを得たい状況です。

【希望】
- 1 件あたりの調査見積もり
- 想定 5-10 件 (上記とは別の特許含む)
- 報告書形式 (法的判断の明示)

ご対応の可否、ご見積もりをお願いいたします。
```

## 弁理士の見つけ方

1. **日本弁理士会** ( https://www.jpaa.or.jp/ ) - 検索可能
2. **弁理士.com** ( https://benrishi.com/ ) - 専門分野で絞り込み
3. **クラウド系**: アイビーマート、CoCoMite 弁理士 (オンライン)
4. **企業向け**: 大手 - 中村合同特許法律事務所、TMI 等

**おすすめ**: 個人発明家 / 中小企業の特許に詳しい弁理士
(価格が手頃で、本テーマと相性良い)

## Phase 1 → Phase 2 への移行基準

```
GO 条件:
- 5-10 件中、3 件以上が完全クリア
- 弁理士から「量産販売 OK」の判断
- 上記 3 件の中に、市場需要が確認できるもの 1+ 件
- 試作費 + 初回ロット金額 (¥30-100K) を投資可能

NG 条件:
- 全件で改良発明や商標が現役 → 候補リスト再生成
- 弁理士の判断が「グレー」 → 別候補へ移行
- 米国・中国でアマゾン特許訴訟事例 → スキップ
```

## Phase 1 後の自動化拡張

弁理士確認の結果を JSON で保存し、score JSON にマージ:

```json
{
  "patent_number": "JP2003-329421A",
  "scores": {...},
  "legal_clearance": {
    "patent_status": "expired",
    "design_patent": "none",
    "trademark": "none",
    "improvement_patent": "none",
    "us_jp_overlap": "checked_clean",
    "verdict": "CLEAR",
    "reviewed_by": "○○弁理士",
    "reviewed_at": "2026-05-15",
    "fee_jpy": 25000,
    "report_url": "..."
  }
}
```

これで `automation/patent_mine/legal_filtered.json` を生成し、
**Phase 2 (試作・量産) に進むのは CLEAR 案件のみ**とする。
