# 投稿戦略ドキュメント

## ブランド構造

```
AIコンパス（親ブランド）
  ├─ 既存ファネル記事（毎朝10:01投稿）
  │   業界別AI活用事例（廃棄物処理、農業、製造業など）
  │   タイトル形式: 「【業界】の【課題】をAI化｜【数値】削減の方法」
  │
  └─ 【新連載】AIで投資の壁を越える（毎週水曜12:00投稿）
       6ヶ月かけて6つの壁を実装で検証
       タイトル形式: 「【AIで投資の壁を越える #N】xxxxx」
```

## なぜこの2本立てが機能するか

### 既存ファネル記事の役割
- **集客**: 業界別の具体的事例で幅広い層を取り込む
- **信頼**: 数値訴求型でブランド信頼性を構築
- **SEO**: 業界別キーワードで検索流入を稼ぐ

### 新連載の役割
- **深耕**: 業界紹介で集めた読者を「AIの限界探求」に引き込む
- **コミュニティ**: 連載読者は熱量が高く、長期フォロワーになる
- **思想**: AIコンパスの「方位磁針」というブランド本質を体現
- **収益**: 将来的に有料マガジン化、書籍化、講演などへの土台

### クロスファネル設計

```
新規読者
  ↓ Google検索 / X / 知人紹介
  ↓
既存ファネル記事（朝10:01）
  ↓ 「もっと深い内容を読みたい」層
  ↓
新連載（水曜12:00）
  ↓ 6ヶ月の連載で深いファン化
  ↓
将来の収益化（有料マガジン、書籍、講演）
```

## マガジン機能の活用

### マガジン1: 「AIで投資の壁を越える」（新規作成）
- 全連載記事をまとめる
- 連載専用のサイドバー的な機能
- 過去記事への遡及流入を確保

### マガジン2: 「業界別AI活用事例」（既存）
- ファネル記事をまとめる
- 既に運用中のはず

### 相互リンク
- 各マガジン説明文で互いをリンク
- 記事内でも互いに参照

## エンゲージメント施策

### 連載読者を増やす施策

1. **第0回（マニフェスト）の徹底告知**
   - X で連投スレッド
   - 既存ファネル記事の末尾に告知追加
   - フォロワーへの個別告知

2. **第1回のインパクト最大化**
   - 「年利101%が10年バックテストで13%」という結論
   - 表とグラフを画像化してX投稿
   - 引用RT を活用

3. **継続性の演出**
   - 毎週水曜の更新を絶対に守る
   - X日次速報で「次回予告」を流す
   - 連載タグで一覧性を保つ

### 想定される批判への対応

| 批判 | 対応 |
|------|------|
| 「投資助言ではないか」 | 免責事項を毎記事明記、シミュレーション宣言 |
| 「個人攻撃ではないか」 | 全て匿名化、戦略のみ評価 |
| 「結局儲かるの？」 | 儲けが目的ではない旨を毎回確認 |
| 「日本株は？」 | 第3-4回以降で日本株シリーズも展開 |
| 「クリプトは？」 | 暗号通貨専門回も計画中 |
